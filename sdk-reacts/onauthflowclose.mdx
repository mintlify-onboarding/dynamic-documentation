---
title: "OnAuthFlowClose"
---

### Summary

This callback will be called after the auth flow modal is closed.

### Usage

```TypeScript
<DynamicContextProvider
  settings={{
    eventsCallbacks: {
      onAuthFlowClose: () => {
        console.log('onAuthFlowClose was called');
      }
    }
  }}
>
 {/* ... rest of your app ... */}
</DynamicContextProvider>
```

### Callback Argument Value

None
