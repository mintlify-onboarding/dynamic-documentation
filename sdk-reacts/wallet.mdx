---
title: "Wallet"
description:
  "An object containing the attributes about the connected wallet. This object
  would be available immediately after a wallet has been connected."
---

| Field                      | Description                                   |
| -------------------------- | --------------------------------------------- |
| address: string            | Public key/address of the connected wallet    |
| chain: string              | Current BlockChain name                       |
| connected: boolean         | Whether this wallet has been connected or not |
| connector: WalletConnector | See WalletConnector object above              |
